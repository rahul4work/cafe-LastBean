# Cafe LastBean ☕️

A beautiful and responsive frontend website for a coffee shop, built with HTML, CSS, and JavaScript.

## 🌟 Features

- Responsive design for mobile, tablet, and desktop
- Interactive menu navigation
- Smooth scroll
- Swiper.js slider
- Clean, modern UI

## 🚀 Tech Stack

- HTML5
- CSS3
- JavaScript
- Swiper.js
